# openvpn 文档

目前 ubuntu 22.04 支持 openvpn 2.5.9
openvpn 社区刚发布 2.6.8
未来至少一年内可能会使用 openvpn release 2.6

## openvpn [服务端配置](https://openvpn.net/community-resources/reference-manual-for-openvpn-2-6/)

这份配置都是英文的，不太好让中文用户理解，为了方便后续维护、对接，需要整理一份中文服务端配置文档。

英文文档是基于 `python-docutils` 操作 `openvpn.8.rst` 文件生成的。
中文文档是基于 `python-docutils` 操作 `openvpn.8.cn.rst` 文件生成的。
